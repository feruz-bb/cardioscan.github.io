Qanday Hal Qilamiz? Texnologik Yondashuv

Asosiy Vositalar va Stack

  • Front-end: React, Tailwind CSS (Responsiv dizayn)
  • Back-end: Python (Flask/Django)
  • Database: Google Cloud Firestore (Real-time va Xavfsiz saqlash)
  • AI/ML: Scikit-learn, Pandas (Gradient Boosting)
  • Generative AI: Gemini API (Shaxsiylashtirilgan tavsiyalar)

AI Texnologiyalaridan Foydalanish

1. Xavfni Tasniflash Modeli

AI bemorning 8-12 ta asosiy ma'lumotlarini (yosh, chekish, bosim) tahlil qilib, 0 dan 100 gacha bo'lgan Xavf Indeksini taqdim etadi.

2. Tushuntirish (Explainable AI - XAI)

SHAP kabi usullardan foydalanib, AI ning qaysi omillarni eng muhim deb hisoblaganini ko'rsatamiz. Bu natijalarga ishonchni oshiradi.

3. Shaxsiylashtirilgan Tavsiyalar

Generative AI (Gemini API) yordamida Xavf Indeksiga asoslangan holda, oziqlanish, jismoniy faollik va stressni boshqarish bo'yicha shaxsiy tavsiyalar matni yaratiladi.
